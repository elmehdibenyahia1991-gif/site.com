# StoreForge AR - منصة إنشاء المتاجر الإلكترونية

واجهة عربية جاهزة لبيع خدمة إنشاء المتاجر الإلكترونية للزبناء مع تفعيل الدفع عبر PayPal.

## ماذا يقدّم المشروع؟

- عرض باقات الخدمة (49$ / 99$ / 199$).
- توليد بطاقة طلب متجر للزبون مباشرة.
- أزرار دفع PayPal (لكل باقة + زر ديناميكي لطلب خاص).
- إرسال تفاصيل الطلب عبر البريد للعميل ولصاحب المنصة.

## تشغيل محلي

```bash
python3 -m http.server 4173
```

ثم افتح:

`http://127.0.0.1:4173/index.html`

## تفعيل PayPal (عمليًا)

1. ادخل إلى حساب PayPal Developer.
2. أنشئ App وخذ `Client ID`.
3. داخل الصفحة نفسها أدخل `Client ID` واضغط **تفعيل PayPal الآن**.
4. سيحفظ الإعداد في `localStorage` لنفس المتصفح.

> لا حاجة لتعديل الكود يدويًا بعد الآن.

## إخراج الموقع لحيز التطبيق (Deployment)

### الخيار 1: Netlify (الأسهل)
1. ارفع هذا المستودع إلى GitHub.
2. من Netlify: **Add new site → Import from Git**.
3. اختر المستودع.
4. Build command: *(فارغ)*
5. Publish directory: `.`
6. Deploy.

### الخيار 2: GitHub Pages
1. ادخل إلى إعدادات المستودع على GitHub.
2. Pages → Source = Deploy from a branch.
3. اختر الفرع `main` (أو الفرع المطلوب) والمجلد `/ (root)`.
4. احفظ الإعداد وانتظر رابط الموقع.
