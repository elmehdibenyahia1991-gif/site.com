# StoreForge AR - منصة إنشاء المتاجر الإلكترونية

واجهة عربية/فرنسية جاهزة لبيع خدمة إنشاء المتاجر الإلكترونية مع أزرار دفع PayPal مباشرة.

## المزايا

- عرض 3 باقات واضحة: `49$ / 99$ / 199$`.
- أزرار دفع PayPal مفعّلة لكل باقة (تحويل مباشر إلى PayPal).
- دعم لغتين (AR/FR) مع RTL/LTR.
- قسم تواصل مباشر عبر البريد:
  - `Elmehdibenyahia1991@gmail.com`

## تشغيل محلي

```bash
python3 -m http.server 4173
```

ثم افتح:

`http://127.0.0.1:4173/index.html`

## ملاحظات الدفع

- تم إعداد الدفع عبر نموذج PayPal القياسي باستخدام البريد:
  - `elmehdibenyahia1991@gmail.com`
- يمكنك لاحقًا ترقية التكامل إلى PayPal Smart Buttons عبر Client ID إذا رغبت.
